# 学生成绩预测 API（LightGBM + FastAPI）

本项目提供一个基于 **LightGBM 多输出回归模型** 的学生成绩预测服务。
通过输入学生学习行为与学习态度特征，预测其：

* 数学成绩（数值 + 等级）
* 阅读成绩（数值 + 等级）
* 科学成绩（数值 + 等级）

成绩等级自动按照分数区间映射为：

| 分数区间  | 等级 |
| ----- | -- |
| ≥80   | 优秀 |
| 60–79 | 良好 |
| 40–59 | 中等 |
| 20–39 | 较差 |
| <20   | 差  |

适用于：

* 学习能力画像生成
* 教育系统冷启动建模
* 推荐系统用户初始化特征估计
* 学习路径规划 Agent

# 接口说明

## POST /predict_score

用于预测学生三科成绩（包含数值结果 + 等级标签）。

请求格式：

```
application/json
```

请求示例：

```json
{
  "gender": "男",
  "grade": "初中",
  "daily_study_time": "1小时以上",
  "math_recognition": "我认为数学十分有用且热爱学习数学",
  "learning_autonomy": "我高度自主且有强烈学习意愿",
  "learning_perseverance": "高度坚持，主动克服各种困难",
  "learning_curiosity": "高度好奇，热爱学习与探索未知",
  "current_goal": "重点高中"
}
```

返回示例：

```
学生信息：
性别：男
年级：初中
每天学习时长：1小时以上
数学认可：我认为数学十分有用且热爱学习数学
学习自主性：我高度自主且有强烈学习意愿
学习坚持性：高度坚持，主动克服各种困难
学习好奇心：高度好奇，热爱学习与探索未知
目前目标：重点高中

预测成绩：
数学成绩：82.35分 【优秀】
阅读成绩：78.12分 【良好】
科学成绩：80.47分 【优秀】
```

# 输入字段说明

接口接收以下字段：

| 字段名                   | 含义     | 是否参与模型预测 |
| --------------------- | ------ | -------- |
| gender                | 性别     | 是        |
| grade                 | 年级     | 否        |
| daily_study_time      | 每天学习时长 | 是        |
| math_recognition      | 数学认可   | 是        |
| learning_autonomy     | 学习自主性  | 是        |
| learning_perseverance | 学习坚持性  | 是        |
| learning_curiosity    | 学习好奇心  | 是        |
| current_goal          | 当前目标   | 否        |

说明：

以下字段：

```
grade
current_goal
```

目前仅作为用户画像展示字段，不参与模型预测计算。

# 可选输入值范围（受模型约束字段）

以下字段必须严格使用训练时一致的文本，否则可能导致编码失败或预测异常。

---

## gender（性别）

```
男
女
```

---

## daily_study_time（每天学习时长）

```
1小时以上
30分钟-1小时
小于30分钟
```

---

## math_recognition（数学认可）

```
我认为数学十分有用且热爱学习数学
我认为数学比较有用且有学习兴趣
我认为数学有一定作用且兴趣一般
我认为数学作用不大且兴趣很低
我认为数学没用且完全不想学数学
```

---

## learning_autonomy（学习自主性）

```
我高度自主且有强烈学习意愿
我能自主规划并坚持学习
我能主动完成学习任务
我会主动学一点
我几乎没有动力学习
```

---

## learning_perseverance（学习坚持性）

```
高度坚持，主动克服各种困难
面对挑战能坚持，不易放弃
能完成基本任务，遇到困难可坚持
遇到困难容易退缩，坚持度较低
遇到困难立刻放弃，几乎不能坚持
```

---

## learning_curiosity（学习好奇心）

```
高度好奇，热爱学习与探索未知
好奇心较强，喜欢主动学习探索
对学习有一定兴趣，愿意了解新知
对学习兴趣较低，很少主动探索
对学习毫无兴趣，不想探索新知识
```

# 模型说明

模型类型：

```
LightGBM 多输出回归模型
```

输入特征：

```
gender
daily_study_time
math_recognition
learning_autonomy
learning_perseverance
learning_curiosity
```

输出：

```
[数学成绩, 阅读成绩, 科学成绩]
```

附加处理：

模型预测结果会自动转换为能力等级：

```
优秀 / 良好 / 中等 / 较差 / 差
```

编码方式：

```
encoder.pkl
```

用于分类变量数值转换。

模型文件：

```
lgb_model.pkl
```

---

# 注意事项

以下字段不会影响预测结果：

```
grade
current_goal
```

它们仅用于：

* 用户画像展示
* 推荐系统扩展输入
* 后续模型升级预留字段

如需参与预测，需要重新训练模型。
